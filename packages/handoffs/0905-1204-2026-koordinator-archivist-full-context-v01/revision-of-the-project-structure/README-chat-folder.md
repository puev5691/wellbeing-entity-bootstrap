# Рабочий пакет чата: revision-of-the-project-structure

Статус: активный рабочий пакет
Назначение: ревизия структуры проекта, подготовка правил пакетизации чатов, проектирование и первичная реализация MVP Архивариуса проекта Благополучие

## Что лежит в этом каталоге

В этом каталоге хранятся:
- проектные документы по Архивариусу;
- журналные записи этапов;
- shell-скрипты генерации регламентов и вспомогательных файлов;
- результаты промежуточной ревизии;
- материалы, созданные в ходе работы текущего чата.

## Роль каталога

Этот каталог является рабочим пакетом одного чата.

Он используется как:
- контейнер происхождения материалов;
- мастер-пакет темы;
- локальный архив работы по ревизии структуры проекта;
- реальный тестовый пакет для обкатки Архивариуса на живых данных.
